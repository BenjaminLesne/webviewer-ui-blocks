## How It Works

The components share zoom state through React context. When you click `+` or `-` on either control:

1. The component calls `zoomTo()` from the `useZoom` hook
2. The hook updates the `documentViewer` zoom level
3. All subscribed components receive the `zoomUpdated` event
4. Each component re-renders with the new zoom level

This architecture means you can place zoom controls anywhere in your app and they'll automatically stay synchronized.

## Code

Here's the code for the demo above:

```tsx
import { LoadWebviewerCore } from '@/components/load-webviewer-core';
import { LoadPDFNet } from '@/components/load-pdfnet';
import { WebViewerProvider } from '@/context/webviewer-context';
import { WebViewerUI } from '@/components/webviewer-ui';
import { ViewerCanvas } from '@/components/viewer-canvas';
import { Document } from '@/components/document';

export function PDFViewer() {
  return (
    <LoadWebviewerCore>
      <LoadPDFNet>
        <WebViewerProvider>
          <WebViewerUI title="W-9 Form">
            <ViewerCanvas className="h-100 overflow-auto">
              <Document documentPath="/forms/w9.pdf" />
            </ViewerCanvas>
          </WebViewerUI>
        </WebViewerProvider>
      </LoadPDFNet>
    </LoadWebviewerCore>
  );
}
```
