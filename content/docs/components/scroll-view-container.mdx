The `ScrollViewContainer` component creates the scrollable container for the PDF viewer. For most use cases, use `ViewerCanvas` instead, which combines this with `ViewerElement`.

## Installation

<InstallCommand name="scroll-view-container" />

## Usage

```tsx
import { ScrollViewContainer } from '@/components/scroll-view-container';
import { ViewerElement } from '@/components/viewer-element';

function CustomViewer() {
  return (
    <ScrollViewContainer className="h-screen overflow-auto">
      <ViewerElement className="min-h-full" />
    </ScrollViewContainer>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Child components (typically ViewerElement) |
| `className` | `string` | - | CSS class names for the scroll container |
| `style` | `CSSProperties` | - | Inline styles for the scroll container |

## Notes

- Must be used within a `WebViewerProvider` context
- Calls `documentViewer.setScrollViewElement()` on mount
- Cleans up by setting element to `null` on unmount
- Handles scroll events for the PDF viewer
