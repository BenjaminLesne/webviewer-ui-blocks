The `ViewerCanvas` component is the main component for displaying PDF documents. It combines `ScrollViewContainer` and `ViewerElement` into a single, easy-to-use component.

## Installation

<InstallCommand name="viewer-canvas" />

## Usage

```tsx
import { LoadWebviewerCore } from '@/components/load-webviewer-core';
import { LoadPDFNet } from '@/components/load-pdfnet';
import { WebViewerProvider } from '@/context/webviewer-context';
import { ViewerCanvas } from '@/components/viewer-canvas';
import { Document } from '@/components/document';

export function PDFViewer() {
  return (
    <LoadWebviewerCore>
      <LoadPDFNet>
        <WebViewerProvider>
          <ViewerCanvas className="h-screen w-full overflow-auto">
            <Document documentPath="/sample.pdf" />
          </ViewerCanvas>
        </WebViewerProvider>
      </LoadPDFNet>
    </LoadWebviewerCore>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | CSS class names for the scroll container |
| `style` | `CSSProperties` | - | Inline styles for the scroll container |
| `children` | `ReactNode` | - | Content (typically Document component) |

## Notes

- Must be used within a `WebViewerProvider` context
- Combines `ScrollViewContainer` (outer) and `ViewerElement` (inner)
- The `Document` component should be passed as a child
- For advanced layouts, use `ScrollViewContainer` and `ViewerElement` separately
